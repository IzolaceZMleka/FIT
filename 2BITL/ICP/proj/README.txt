Projekt do předmětu ICP
Aplikace Othello (Reversi), která vychází ze stejnojmenné deskové hry.

Autoři: Adam Piecek (xpiece00), Jan Pavlica (xpavli78)

poznámky:
funkční - gui, command line, hra pro dva hráče, hra s umělou inteligencí, undo, redo
nefunční - načítání hry

EDIT 26.5.2016:
- načítání hry opraveno, je funkční
- program využívá boost knihovny pro serializaci
